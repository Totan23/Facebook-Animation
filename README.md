# Facebook Animation

Esta aplicación iOS demuestra una animación interactiva inspirada en las reacciones de Facebook. Al tocar la pantalla, se generan múltiples iconos animados (corazones y "me gusta") que aparecen y se desplazan siguiendo una trayectoria curva con animaciones suaves tipo Bézier. Los iconos varían aleatoriamente en tamaño y duración de la animación, creando un efecto visual dinámico y entretenido similar al sistema de reacciones de las redes sociales.
